## 🐧 Overview

A highly customizable desktop shell built on Quickshell, featuring configurable widgets, bar, dock, launcher, powermenu, and OSD — forked from Axenide/Ax-Shell.

---

## 🛠️ Features

- Modular widget system (clock, system tray, workspace, etc.)
- Custom dock and launcher
- Power management menu
- On-screen display (OSD)
- QML + JavaScript frontend with Python helpers
